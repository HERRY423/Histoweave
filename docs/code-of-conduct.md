# Code of Conduct

HistoWeave adopts the **Contributor Covenant 2.1**.

The canonical text is at the repository root:

**[CODE_OF_CONDUCT.md](https://github.com/histoweave-spatial/histoweave/blob/main/CODE_OF_CONDUCT.md)**

## Summary

* We pledge a harassment-free community for everyone.
* Be empathetic, respectful, and professional.
* Unacceptable behavior includes harassment, trolling, doxxing, and sexualized
  language or imagery.
* Enforcement ladder: Correction → Warning → Temporary ban → Permanent ban.

## Report an incident

Email **conduct@histoweave-spatial.org**. All reports are reviewed promptly and
fairly; reporter privacy is respected.

See also: [Contributing](contributing.md).
